Overview

In this program you will implement a main function that contains Wordleâ€™s â€œgame loopâ€. The game loop is what controls the overall game logic:

The main flow of the game works as follows:

  1. You have up to six turns
  2. Each turn the player gets to input_guess of the same length of each word
